# AutoGen Patterns and Best Practices

## Core Concepts

### Conversable Agents
Agents that can communicate with each other through messages.

### Group Chat
Multi-agent conversations with orchestration.

### Code Execution
Built-in support for running Python code in safe environments.

### Human-in-the-Loop
First-class support for human participation in agent workflows.

## Agent Types

### Assistant Agent

**Pattern:** AI-powered agent that can generate responses and call functions

**Example:**
```python
from autogen import AssistantAgent

assistant = AssistantAgent(
    name="assistant",
    llm_config={
        "model": "gpt-4",
        "temperature": 0,
        "api_key": os.environ["OPENAI_API_KEY"]
    },
    system_message="You are a helpful AI assistant."
)
```

**Characteristics:**
- ✅ Can use tools/functions
- ✅ Generates code
- ❌ Cannot execute code (needs UserProxyAgent)

### User Proxy Agent

**Pattern:** Represents user and can execute code

**Example:**
```python
from autogen import UserProxyAgent

user_proxy = UserProxyAgent(
    name="user_proxy",
    human_input_mode="NEVER",  # No human input
    code_execution_config={
        "work_dir": "coding",
        "use_docker": False  # Set True for isolated execution
    },
    max_consecutive_auto_reply=10
)
```

**Human Input Modes:**
- `NEVER`: Fully autonomous
- `TERMINATE`: Asks for input on termination
- `ALWAYS`: Asks for input on every message

**Characteristics:**
- ✅ Executes code generated by AssistantAgent
- ✅ Can represent human users
- ✅ Acts as conversation initiator

### Groupchat Manager

**Pattern:** Orchestrates multi-agent conversations

**Example:**
```python
from autogen import GroupChat, GroupChatManager

groupchat = GroupChat(
    agents=[assistant1, assistant2, user_proxy],
    messages=[],
    max_round=10
)

manager = GroupChatManager(
    groupchat=groupchat,
    llm_config={"model": "gpt-4"}
)
```

## Conversation Patterns

### Two-Agent Pattern

**Pattern:** Simple back-and-forth between two agents

**Example:**
```python
assistant = AssistantAgent(name="assistant", llm_config=config)
user_proxy = UserProxyAgent(name="user", code_execution_config={...})

# Initiate conversation
user_proxy.initiate_chat(
    assistant,
    message="Write a function to calculate fibonacci numbers"
)
```

**Flow:**
1. User proxy sends request
2. Assistant generates code
3. User proxy executes code
4. Results sent back to assistant
5. Repeat until task complete

### Multi-Agent Debate

**Pattern:** Multiple agents discuss to reach consensus

**Example:**
```python
# Create multiple assistants with different personas
researcher = AssistantAgent(
    name="Researcher",
    system_message="You are a research expert who values accuracy and evidence.",
    llm_config=config
)

critic = AssistantAgent(
    name="Critic",
    system_message="You are a critical thinker who questions assumptions.",
    llm_config=config
)

synthesizer = AssistantAgent(
    name="Synthesizer",
    system_message="You synthesize different viewpoints into conclusions.",
    llm_config=config
)

user_proxy = UserProxyAgent(name="User", human_input_mode="NEVER")

# Group chat for debate
groupchat = GroupChat(
    agents=[researcher, critic, synthesizer, user_proxy],
    messages=[],
    max_round=15,
    speaker_selection_method="round_robin"  # Or "auto"
)

manager = GroupChatManager(groupchat=groupchat, llm_config=config)

user_proxy.initiate_chat(
    manager,
    message="Is AI going to replace software developers?"
)
```

**Speaker Selection Methods:**
- `round_robin`: Each agent speaks in turn
- `auto`: LLM decides who speaks next
- `manual`: User selects speaker
- `random`: Random selection

### Sequential Handoff

**Pattern:** Explicit agent-to-agent handoffs

**Example:**
```python
# Agent 1: Research
researcher.initiate_chat(
    user_proxy,
    message="Research the top 3 Python web frameworks"
)

# Get result
research_result = user_proxy.last_message()["content"]

# Agent 2: Analysis (uses research result)
analyst.initiate_chat(
    user_proxy,
    message=f"Analyze these frameworks: {research_result}"
)
```

### Nested Chats

**Pattern:** Agent initiates sub-conversation with other agents

**Example:**
```python
from autogen import initiate_chats

# Define nested conversations
chat_queue = [
    {
        "recipient": research_agent,
        "message": "Research topic X",
        "summary_method": "last_msg"
    },
    {
        "recipient": writer_agent,
        "message": "Write based on research",
        "summary_method": "reflection_with_llm"
    }
]

# Execute sequentially
results = initiate_chats(chat_queue)
```

## Code Execution Patterns

### Safe Code Execution with Docker

**Pattern:** Execute code in isolated Docker container

**Example:**
```python
user_proxy = UserProxyAgent(
    name="user",
    code_execution_config={
        "work_dir": "workspace",
        "use_docker": True,
        "timeout": 60,
        "image": "python:3.11-slim"  # Custom Docker image
    }
)
```

**Benefits:**
- ✅ Isolated execution (no system access)
- ✅ Reproducible environment
- ❌ Slower (Docker overhead)

### Local Code Execution

**Pattern:** Execute code directly on host (faster, less safe)

**Example:**
```python
user_proxy = UserProxyAgent(
    name="user",
    code_execution_config={
        "work_dir": "workspace",
        "use_docker": False,
        "timeout": 30
    }
)
```

**Use cases:**
- Quick prototyping
- Trusted environments
- Performance-critical applications

**Warning:** ⚠️ Code runs with your permissions - be careful!

### Code Execution Guards

**Pattern:** Review code before execution

**Example:**
```python
def is_code_safe(code: str) -> bool:
    """Basic safety check"""
    dangerous = ["os.system", "subprocess", "eval", "exec", "__import__"]
    return not any(d in code for d in dangerous)

user_proxy = UserProxyAgent(
    name="user",
    code_execution_config={
        "work_dir": "workspace",
        "use_docker": False
    },
    is_termination_msg=lambda msg: "TERMINATE" in msg.get("content", ""),
    code_execution_config={
        "last_n_messages": 1,
        "work_dir": "workspace",
        "use_docker": False
    }
)

# Custom execution function
def safe_execute_code(code: str):
    if is_code_safe(code):
        return user_proxy.execute_code(code)
    else:
        return "Code execution blocked: potentially unsafe"
```

## Function Calling

### Registering Functions

**Pattern:** Give agents access to custom functions

**Example:**
```python
from autogen import register_function

def search_database(query: str) -> str:
    """Search database for relevant records."""
    # Implementation
    return results

# Register function with agents
register_function(
    search_database,
    caller=assistant,  # Can call the function
    executor=user_proxy,  # Executes the function
    name="search_database",
    description="Search the database for records matching the query"
)

# Agent can now call search_database
assistant.initiate_chat(
    user_proxy,
    message="Find all users registered in 2024"
)
```

### Function with Multiple Agents

**Example:**
```python
# Multiple agents can call the same function
register_function(
    search_database,
    caller=[assistant1, assistant2],  # Both can call
    executor=user_proxy,  # One executes
    description="Search database"
)
```

### Structured Function Outputs

**Pattern:** Use Pydantic for type safety

**Example:**
```python
from pydantic import BaseModel
from typing import List

class SearchResult(BaseModel):
    id: int
    name: str
    score: float

def search_database(query: str) -> List[SearchResult]:
    """Search database with structured output."""
    results = db.search(query)
    return [SearchResult(**r) for r in results]
```

## Human-in-the-Loop Patterns

### Always Ask for Input

**Pattern:** Human validates every step

**Example:**
```python
user_proxy = UserProxyAgent(
    name="user",
    human_input_mode="ALWAYS",
    code_execution_config={...}
)

# User will be prompted at each step:
# "Press enter to continue, 'exit' to stop, or provide feedback"
```

### Ask on Termination

**Pattern:** Human confirms final result

**Example:**
```python
user_proxy = UserProxyAgent(
    name="user",
    human_input_mode="TERMINATE",
    is_termination_msg=lambda msg: "APPROVE" in msg.get("content", ""),
)

# Only asks for input when agent wants to terminate
```

### Custom Human Input Function

**Pattern:** Advanced input handling

**Example:**
```python
def custom_input_function(prompt: str) -> str:
    """Custom input with logging and validation."""
    print(f"[SYSTEM] {prompt}")
    user_input = input("> ")

    # Log all human inputs
    log_human_input(user_input)

    # Validate
    if user_input.strip() == "":
        return "Please provide a valid response"

    return user_input

user_proxy = UserProxyAgent(
    name="user",
    human_input_mode="ALWAYS",
    get_human_input=custom_input_function
)
```

### Approval Workflow

**Pattern:** Human approves before execution

**Example:**
```python
class ApprovalAgent(UserProxyAgent):
    def __init__(self, **kwargs):
        super().__init__(**kwargs)

    def execute_code_blocks(self, code_blocks):
        """Override to require approval"""
        print("\n=== CODE TO EXECUTE ===")
        for block in code_blocks:
            print(block[1])
        print("======================\n")

        approval = input("Approve execution? (yes/no): ")
        if approval.lower() == "yes":
            return super().execute_code_blocks(code_blocks)
        else:
            return "Execution cancelled by user"

approver = ApprovalAgent(
    name="approver",
    code_execution_config={...}
)
```

## Group Chat Orchestration

### Round Robin

**Pattern:** Each agent speaks in order

**Example:**
```python
groupchat = GroupChat(
    agents=[planner, coder, tester, user_proxy],
    messages=[],
    max_round=20,
    speaker_selection_method="round_robin"
)
```

**Use cases:**
- Predictable workflows
- Each agent has clear role
- Sequential processing

### Auto Selection

**Pattern:** LLM decides who speaks next

**Example:**
```python
groupchat = GroupChat(
    agents=[researcher, analyst, writer, user_proxy],
    messages=[],
    max_round=30,
    speaker_selection_method="auto"
)

manager = GroupChatManager(
    groupchat=groupchat,
    llm_config={"model": "gpt-4"}  # Needs strong model
)
```

**Use cases:**
- Dynamic conversations
- Collaborative problem-solving
- Emergent workflows

**Warning:** Can be unpredictable and expensive

### Custom Selection Function

**Pattern:** Programmatic speaker selection

**Example:**
```python
def custom_speaker_selection(last_speaker, groupchat):
    """Custom logic for selecting next speaker"""
    messages = groupchat.messages

    # After researcher, always go to analyst
    if last_speaker.name == "Researcher":
        return groupchat.agent_by_name("Analyst")

    # After 3 rounds of debate, go to synthesizer
    if len(messages) > 6:
        return groupchat.agent_by_name("Synthesizer")

    # Default
    return groupchat.agent_by_name("User")

groupchat = GroupChat(
    agents=agents,
    messages=[],
    max_round=20,
    speaker_selection_method=custom_speaker_selection
)
```

### Conditional Termination

**Pattern:** End conversation based on conditions

**Example:**
```python
def is_conversation_complete(msg) -> bool:
    """Check if conversation should end"""
    content = msg.get("content", "").lower()

    # End if consensus reached
    if "consensus:" in content:
        return True

    # End if all agents agree
    if "all agree" in content:
        return True

    return "TERMINATE" in content

groupchat = GroupChat(
    agents=agents,
    messages=[],
    max_round=50,
    is_termination_msg=is_conversation_complete
)
```

## Cost Optimization

### Use Cheaper Models for Simple Agents

**Example:**
```python
# Expensive agent for complex reasoning
strategist = AssistantAgent(
    name="strategist",
    llm_config={"model": "gpt-4", "temperature": 0.7}
)

# Cheap agent for simple tasks
formatter = AssistantAgent(
    name="formatter",
    llm_config={"model": "gpt-3.5-turbo", "temperature": 0}
)
```

### Limit Conversation Rounds

**Example:**
```python
groupchat = GroupChat(
    agents=agents,
    messages=[],
    max_round=10  # Prevent runaway conversations
)
```

### Cache LLM Responses

**Example:**
```python
from autogen import Cache

# Enable caching
with Cache.disk() as cache:
    assistant = AssistantAgent(
        name="assistant",
        llm_config={
            "model": "gpt-4",
            "cache": cache  # Reuse responses for identical prompts
        }
    )

    user_proxy.initiate_chat(assistant, message="Hello")
```

### Monitor Token Usage

**Example:**
```python
import tiktoken

def count_tokens(messages, model="gpt-4"):
    """Count tokens in conversation"""
    encoding = tiktoken.encoding_for_model(model)
    tokens = sum(len(encoding.encode(m["content"])) for m in messages)
    return tokens

# After conversation
total_tokens = count_tokens(groupchat.messages)
cost = total_tokens * 0.00003  # Rough GPT-4 cost estimate
print(f"Total tokens: {total_tokens}, Estimated cost: ${cost:.4f}")
```

## Common Anti-Patterns

### ❌ Too Many Agents in Group Chat

**Problem:**
```python
# 10 agents - chaos and high cost
groupchat = GroupChat(agents=ten_agents, max_round=100)
```

**Solution:** Keep to 3-5 agents maximum

### ❌ No Termination Conditions

**Problem:**
```python
# Can run forever
groupchat = GroupChat(agents=agents, max_round=1000)
```

**Solution:**
```python
groupchat = GroupChat(
    agents=agents,
    max_round=20,  # Reasonable limit
    is_termination_msg=lambda msg: "TERMINATE" in msg.get("content", "")
)
```

### ❌ Executing Untrusted Code

**Problem:**
```python
user_proxy = UserProxyAgent(
    name="user",
    code_execution_config={"use_docker": False}  # Dangerous!
)
```

**Solution:** Always use Docker in production
```python
user_proxy = UserProxyAgent(
    name="user",
    code_execution_config={
        "use_docker": True,
        "timeout": 60
    }
)
```

### ❌ Vague System Messages

**Problem:**
```python
agent = AssistantAgent(
    name="helper",
    system_message="You are helpful"
)
```

**Solution:**
```python
agent = AssistantAgent(
    name="python_expert",
    system_message="""You are a Python expert specializing in data analysis.
    You write clean, efficient code following PEP 8 guidelines.
    Always include error handling and docstrings."""
)
```

## Production Best Practices

1. **Always use Docker** for code execution in production
2. **Set max_round limits** to prevent runaway costs
3. **Monitor token usage** to control expenses
4. **Use specific system messages** for predictable behavior
5. **Implement termination conditions** for every workflow
6. **Log all conversations** for debugging and auditing
7. **Use cheaper models** (GPT-3.5) for simple agents
8. **Test with mock responses** before production deployment
9. **Implement human approval** for critical operations
10. **Set execution timeouts** to prevent hanging
