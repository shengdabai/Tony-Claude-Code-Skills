# /rename — 终端标签页重命名

## 执行

用户输入 `/rename <名称>` 后，执行：

```bash
printf '\e]0;%s\a' '<名称>'
```

然后回复：「终端已命名为 **<名称>**」

## 规则

- 设置后的终端名称在整个会话中保持不变
- **禁止**在后续任何操作中修改终端标题（不得执行任何 `\e]0;` 或 `\e]2;` 转义序列）
- 如果用户没有提供名称，提示用户输入
